Visualizar Compras

Dentro de esta pantalla se podrá observar el listado de todas las compras que se hicieron, a las mismas se les podrá aplicar filtros para poder observar mejor la información plasmada en el listado

Se podrá aplicar 3 filtros, las opciones se podrán ver si se despliega donde dice “Filtrar por”:

  • Por monto: se debe poner dos valores, los mismos hacen referencia al precio de cada una de las compras, de esta forma se podrá filtrar dentro de los valores de precios añadidos.

  • Por proveedor: aparecerá un despegable el cual posee el listado de todos los proveedores, se puede ir seleccionando y ver como se reduce la lista dependiendo de que proveedor se seleccione

  • Por fecha: se elige dos fechas, pueden ser las dos del mismo día, esto hace que se filtre en base a ese rango y mostrar las compras que se hicieron en esas fechas

    • Todos estos filtros sirven para poder leer de mejor manera los datos y si se desea utilizar esa información para un análisis personal.
