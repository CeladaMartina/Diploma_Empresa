Administrar Usuario Familia Patente

Dentro de esta pestaña se podrá agregarle o quitarle a usuarios, familias o patentes. A la derecha se ve un listado que mostrara las relaciones que tiene el usuario seleccionado, la misma va a ir cambiando a medida que se vayan seleccionando diferentes usuarios. En cambio el listado que está debajo muestra todas las relaciones que tienen todos los usuarios con sus respectivas familias y/o patentes, solamente va a modificarse cuando se agreguen o se quiten relaciones

Las relaciones pueden ser: usuario – familia y usuario – patente. Ambas se pueden realizar de manera muy similar:

  1. Se selecciona del despegable de usuario el usuario al que se le quiera relacionar una familia o una patente, una vez seleccionado se cliquea el botón “Seleccionar”

  2. Luego se selecciona del despegable a la familia o a la patente dependiendo de a quien se lo quiera relacionar y después se cliquea el botón “Agregar” que este a la altura de lo que seleccionamos ya sea familia o patente

  3. Se cliquea por último el botón “Guardar” para que quede sentado las relaciones que creamos

Si se quiere quitar una relación el proceso es el mismo que el anterior pero en vez de cliquear el botón “Asignar” se cliquea el botón “Quitar” y de esa forma se borrara la relación
