Administrar Permisos

Dentro de esta pantalla se pueden realizar varias opciones.

Familia

En esta pestañase podrá modificar a las familias

  • Se podrá dar de altas a nuevas familias, esto sucede si se completa solamente el campo nombre, y cliqueando el botón “Crear”, de esta forma se habrá dado de alta a una familia nueva

  • Se podrá visualizar esta nueva familia en el combo de "Todas las familias"

  • Para agregar familias a familias:

    1. Primero se selecciona una familia del despegable familia y se debe apretar el botón "Seleccionar"

    2. Luego del mismo despegable se selecciona otra familia y en vez de apretar el botón "Seleccionar" se cliquea el botón "Agregar" que está a la altura de familia, esto lo que hará es que una familia esté relacionada con una familia

    3. Para agregar patentes a familias:

      1. Primero se selecciona a la familia de la misma forma que arriba, primero se elige del despegable a la familia que queremos y ponemos "Seleccionar"

      2. Del despegable de debajo de patentes se selecciona a la patente a la que queremos relacionar con esa familia

      3. Se aprieta el botón "Agregar" pero el que está a la altura de la patente

      4. Y por último se cliquea el botón "Cargar". De esta forma quedara relacionada la familia con la patente

      5. Para darle de baja a una relación es lo mismo que los pasos anteriores pero en vez de apretar el botón "Agregar" se cliquea el botón "Borrar", y luego el botón "Cargar" de esta forma quedara eliminada esa relación.

      Patente

      En esta pestaña se podrá agregar o quitar patentes del sistema.

    4. Para poder cargarlas primero se debe elegir una opción del desplegable de permisos, ahí se mostraran los permisos que hay en el sistema. Se cliquea el botón “Agregar” y se guardaran los cambios

    5. Para poder quitar una patente del sistema lo único que hay que hacer es seleccionar una opción del despegable de permisos y apretar el botón “Borrar” de esta forma la patente será borrada del sistema

      6. Administrar Permisos

      En esta pestaña se administrara la relación entre las familias con las patentes o familias con familias. El listado que está a la derecha mostrara las relaciones entre familia y patentes, se mostrara en forma descendente mostrando que contiene cada uno, esto se actualizara a medida que se vayan cargando relaciones nuevas y mostrara diferentes relaciones dependiendo de qué familia se seleccione.

    6. Para agregar familias a familias:

      1. Primero se selecciona una familia del despegable familia y se debe apretar el botón “Seleccionar”

      2. Luego del mismo despegable se selecciona otra familia y en vez de apretar el botón “Seleccionar” se cliquea el botón “Agregar” que está a la altura de familia, esto lo que hará es que una familia esté relacionada con una familia

      3. Para guardar esto se debe cliquear el botón “Cargar”

      4. Para borrar relaciones familia - familia, se debe hacer un proceso muy parecido, se hace todo tal cual, la única diferencia que al final se aprieta el botón “Borrar” en vez del que “Agregar” de esta forma la relación familia – familia será borrada

      Sea cual sea la relación que se desea crear o eliminar todos los movimientos se verán reflejados en las dos tablas, en la de debajo de forma permanente, todos los movimientos se verán y en la de la derecha solo se verán las relaciones de la familia seleccionada
