Restore

Dentro de esta pantalla se podrá restaurar un BackUp ya guardado. En la parte superior aparece un listado que esta vacío el mismo se va a llenar cuando elijamos el BackUp que queremos restaurar. Para lograrlo se debe:

  1. Primero se debe cliquear el botón “Agregar” se abrirá una ventana para que elijamos a donde está el BackUp y lo seleccionemos, al darle aceptar se cargara el BackUp.

  2. El botón “Limpiar”, se usa para que si uno se equivoca al seleccionar el BackUp, se pueda otra vez volver a elegir.

  3. Por último se selecciona el botón “Realizar Restore” al cliquearlo después de unos segundos dirá que el restore fue exitoso o no.

Al finalizar con todo, tendremos todos los datos en el sistema ya cargados, en base al BackUp que se hizo
