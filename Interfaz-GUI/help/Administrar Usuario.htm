Administrar Usuario

Usuarios

En esta pestaña se podrá manipular las acciones del usuario. Antes de explicar las funcionalidades, se explicara los datos dentro de esta pestaña

  • Nick: es el identificador del usuario que usara para ingresar al sistema

  • Contraseña: es un valor alfanumérico que se crea aleatoriamente y después se puede cambiar

  • Nombre: del usuario

  • Mail: del usuario

  • Idioma: dentro del despegable se puede ver todos los idiomas del sistema y se debe elegir uno, como predeterminado

    • Las funcionales son:

  • Dar de alta: es para ingresar a un usuario nuevo al sistema, primero se deberá completar todos los campos, el campo de contraseña se completa cliqueando el botón “Generar Contraseña” el mismo creara una contraseña visible que luego el usuario podrá cambiar en la ventana de cambiar contraseña (esto se explica en el documento Seguridad – Generar Contraseña). Luego que se completen todos los campos se debe cliquear el botón “Alta”, de esta forma quedara dado de alta el usuario en el sistema

  • Modificar un usuario: se podrá modificar los datos del usuario excepto la contraseña. Esto será posible si se selecciona de la lista de abajo el usuario que se quiera modificar, los datos se cargaran en los campos, solamente se deberá cambiar los datos que se vean necesarios y luego presionar el botón “Modificar” y el usuario será modificado

  • Dar de baja: se podrá dar de baja a un usuario que no se desea que este más en el sistema, no se lo borrara de la lista sino que solo se dará check a la columna de baja y no podrá ingresar al sistema ni participar de ninguna de todas las operaciones dentro del sistema

  • Desbloquear usuario: este botón se cliqueara cuando el usuario se encuentre bloqueado, es decir que ingreso más de 3 veces mal el Nick o la contraseña o ambos en el logIn, para desbloquearlo lo único que hay que hacer es apretar el botón, y el contador del usuario volverá a 0 y quedara desbloqueado del sistema
